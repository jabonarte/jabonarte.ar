Tipos de Piel

Conocé tu tipo de piel

Cada tipo de piel tiene características únicas y necesita cuidados específicos.

Normal

Características:

  • • Equilibrada y saludable
  • • Poros pequeños
  • • Textura suave
  • • Sin brillos excesivos

Ingredientes recomendados:

Vitamina C Aloe Vera Miel

Seca

Características:

  • • Sensación de tirantez
  • • Descamación
  • • Poros pequeños
  • • Líneas finas tempranas

Ingredientes recomendados:

Ác. Hialurónico Ceramidas Avena

Grasa

Características:

  • • Brillo excesivo
  • • Poros dilatados
  • • Tendencia al acné
  • • Textura gruesa

Ingredientes recomendados:

Ác. Salicílico Arcilla Tea Tree

Mixta

Características:

  • • Zona T grasa
  • • Mejillas secas/normales
  • • Poros variables
  • • Necesita cuidado dual

Ingredientes recomendados:

Niacinamida Jojoba Zinc
